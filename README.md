# vege-family
ベジファミリー - みんなで育てる菜園日記🔰

■ サービス概要
このアプリは、初心者が気軽に家庭菜園を始められることを目指したサービスです。
育てやすい野菜を提案する「ベジ診断」や、種まきから収穫までの手順を示す「ベジ指南」で初心者をサポートします。
さらに、成長記録を写真やメモで共有できる掲示板や、収穫後の料理投稿機能を備え、育てる楽しさを共有できます。
日々の世話や収穫を続けることでスタンプやバッジが獲得でき、継続的なモチベーションを提供します。

■ このサービスへの思い・作りたい理由
きっかけは、知人との何気ない会話からでした。「家の中で育てるならどの野菜が向いているのか？」「何から始めたらいいのか分からない」「どんな野菜が初心者向きなのか」といった相談を受けたのです。私自身、農業関連の業種で働いた経験があり、その知識を活かせば初心者でも気軽に家庭菜園を楽しめるアプリが作れるのではないかと考え、開発を始めました。
野菜を育てる過程には、日々の変化を観察する楽しさや収穫時の達成感があります。例えば、自分で育てたミニトマトを収穫し、その味を家族と一緒に楽しむといった体験は、普通の食事よりも特別な思い出になります。また、これを家族や友人と共有すれば、その楽しさは何倍にもなります。
しかし、野菜を育てるには通常、広大な農地や農機具、肥料、自然災害や害獣対策といった多くの準備が必要です。それに比べ、家庭菜園は畑を持たなくても手軽に始められ、自分のペースで野菜を育てる楽しさが味わえます。
このアプリを通じて、**「自分で育てた野菜を食べてみたい」「子供と一緒に育てる体験をしたい」**そんな思いを多くの人に叶えてもらいたいと考えています。そして私自身も、皆さんと家庭菜園を楽しむことを目指しています。

■ ユーザー層について
* 家庭菜園初心者：何を育てれば良いか迷っている人
* 忙しい社会人：少ない時間で育てたい人
* 家族・友人と楽しみたい人：共同で育てる楽しさを求める人
* 成長記録を残したい人：写真やメモで記録を残したい人

■ サービスの利用イメージ
* ユーザーはまず**「野菜診断」**で育てやすい野菜を選びます。質問に答えるだけで、初心者向けの野菜や家族で育てやすい野菜が提案されます。
* 次に、選んだ野菜の**「育て方指南」**で、種まきから収穫までの具体的なステップが示されます。初心者でも迷わず育成を進められるよう、日々のタスクもサポートされます。
* 育てた野菜は**「成長記録掲示板」**で写真やメモとして記録。他のユーザーや家族と共有することで、育てる楽しさがさらに広がります。
* 収穫後は**「料理投稿」**で成果をシェア。自分で育てた野菜で作った料理を他のユーザーと共有し、反応を楽しむことができます。
* さらに、日々の世話や収穫を続けることで**「スタンプ・バッジ・称号」**が獲得でき、継続のモチベーションを高めます。
